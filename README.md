# novuscap-ecosystem

A design-phase concept and ecosystem map for the **NovusCap** platform. Captures architectural decisions, module boundaries, and the planned feature surface of the ecosystem before implementation.

## Contents

- **NovusCapEkosistemHaritasi** — Visual and code-level ecosystem map outlining service relationships and module structure
- **NovusCapEkosistemHaritasi.sln** — .NET solution entry point
- **package.json** — Tooling for rendering and visualizing the ecosystem map

## Purpose

This repository documents the conceptual design phase: feature scoping, module boundaries, and inter-service dependencies for the NovusCap platform ecosystem.

## Tech Stack

`C#` `.NET` `Node.js`
